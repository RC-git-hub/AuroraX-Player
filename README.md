# AuroraX Player

AuroraX Player is a lightweight Python media player with a modern GUI built using CustomTkinter and MPV. It supports audio & video playback, playlists, shuffle, looping, and fullscreen mode.

## Features
- Audio & video playback using MPV
- Playlists and folder navigation
- Shuffle and loop modes
- Fullscreen toggle (F key or switch)
- Customtkinter dark-themed modern GUI
- Cross-platform support (Windows/macOS/Linux)
- Playback features as YouTube

## Requirements
- Python 3.9+
- `customtkinter`
- `mpv` Python bindings
- `Pillow` (for image handling)
